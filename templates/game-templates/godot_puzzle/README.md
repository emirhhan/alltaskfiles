# Godot Puzzle Template

Godot 4.x (GDScript) ile minimal mantık tabanlı puzzle şablonu: tıklanabilir
kutu ızgarası + basit eşleştirme mantığı + UI geri bildirimi.

## Özellikler
- Grid (ızgara) düğüm yapısı
- Tıklama ile kutu seçimi (signal)
- Basit eşleşme kontrolü

## Çalıştırma
Godot 4.x ile `project.godot` import et, `F5`.
