# Godot Idle / Clicker Template

Godot 4.x (GDScript) ile minimal idle/clicker şablonu: tıkla-kazan para sistemi +
otomatik gelir (idle) + UI gösterimi.

## Özellikler
- Tıklama ile para kazanma
- Zamanlayıcı ile pasif gelir (idle)
- Label ile bakiye gösterimi

## Çalıştırma
Godot 4.x ile `project.godot` import et, `F5`.
