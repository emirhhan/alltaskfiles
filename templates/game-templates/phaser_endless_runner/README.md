# Phaser Endless Runner Template

Phaser 3 (JavaScript) ile minimal endless runner: otomatik ileri kayan dünya +
rastgele engel spawn + çarpışma ile oyun sonu.

## Çalıştırma
```bash
cd templates/game-templates/phaser_endless_runner
npm install
npm run dev      # geliştirme sunucusu (http://localhost:5173)
# veya
npm run build    # production build -> dist/
npm run preview  # build çıktısını önizle
```
