# Godot Endless Runner Template

Godot 4.x (GDScript) ile minimal endless runner: otomatik ileri kayan zemin +
rastgele engel spawn + çarpışma ile oyun sonu.

## Özellikler
- Otomatik dünya kayması (zemin sola akar)
- Timer tabanlı engel spawn
- Engel çarpışmasında oyun bitişi

## Çalıştırma
Godot 4.x ile `project.godot` import et, `F5`.
