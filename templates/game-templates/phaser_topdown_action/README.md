# Phaser Top-Down Action Template

Phaser 3 (JavaScript) ile minimal top-down aksiyon şablonu: 8 yönlü hareket,
düşman kovala AI, hasar/sağlık + invincibility frame.

## Çalıştırma
```bash
cd templates/game-templates/phaser_topdown_action
npm install
npm run dev      # geliştirme sunucusu (http://localhost:5173)
# veya
npm run build    # production build -> dist/
npm run preview  # build çıktısını önizle
```
