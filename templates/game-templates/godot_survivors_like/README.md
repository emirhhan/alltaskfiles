# Godot Survivors-Like Template

Godot 4.x (GDScript) ile minimal survivors-like (dalga + object pooling + düşman)
şablonu. `recipes/godot/spawn_system.yaml` ve `object_pooling.yaml` ile birlikte kullan.

## Özellikler
- Timer tabanlı dalga spawn
- Object pool (önceden üretilmiş düşman havuzu)
- Oyuncu hareketi + basit hasar

## Çalıştırma
Godot 4.x ile `project.godot` import et, `F5`.
