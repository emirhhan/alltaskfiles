# Phaser Survivors-Like Template

Phaser 3 (JavaScript) ile minimal survivors-like: dalga spawn + object pooling +
düşman kovala. `recipes/phaser/spawn_system.yaml` ve `object_pooling.yaml` ile birlikte.

## Çalıştırma
```bash
cd templates/game-templates/phaser_survivors_like
python -m http.server 8082
```
