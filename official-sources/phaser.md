# Phaser — Resmî Kaynaklar

> Kaynak güven puanı (1-5) ve öncelik. Fusion Game bilgi tabanına kayıt için.

| # | Kaynak | URL | Güven | Öncelik | Kullanım |
|---|---|---|---|---|---|
| 1 | Resmî Dokümantasyon | https://docs.phaser.io | 5 | Birincil | API ref, tutorial, example |
| 2 | Phaser Examples | https://phaser.io/examples | 5 | Birincil | Çalışan kod örnekleri |
| 3 | GitHub Repo | https://github.com/phaserjs/phaser | 5 | Birincil | Kaynak kod, issue |
| 4 | Phaser Discord | https://discord.gg/phaser | 4 | Topluluk | Canlı soru-cevap |
| 5 | Phaser Forum | https://phaser.discourse.group | 4 | Topluluk | Tartışma, çözüm |
| 6 | Getting Started (Phaser 3) | https://phaser.io/tutorials/getting-started-phaser3 | 5 | Birincil | Adım adım resmî tutorial |
| 7 | npm paket | https://npmjs.com/package/phaser | 5 | Birincil | Sürüm, tipler |
| 8 | create-phaser-game CLI | https://github.com/phaserjs/phaser | 5 | Birincil | Hızlı proje iskeleti (npm create @phaserjs/game) |

> Not: Phaser 4 (4.2.1) güncel ana sürüm; v3 API'si büyük ölçüde korunur. Proje "Phaser 3" hedefliyse v3 tutorial'ları geçerlidir, v4 için migration guide'a bakılmalı.

## Kaynak Güven Politikası
1. Resmî (docs, examples, GitHub) → seed ders
2. Doğrulanmış topluluk (Discord onaylı) → recipe kaynağı
3. Eğitim → örnek kod
4. Forum → doğrulama gerekir
5. Deneysel → işaretli recipe
